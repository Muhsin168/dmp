# Dynamic Via-points and Improved Spatial Generalization for Online Trajectory Generation with Dynamic Movement Primitives

This package contains the source code used in the simulations and experiments of the paper [**Dynamic Via-points and Improved Spatial Generalization for Online Trajectory Generation with Dynamic Movement Primitives**](https://doi.org/10.1007/s10846-024-02051-0)

- For the simulations, see the instructions in `simulations/README.md`
- For the experiments,  see the instructions in `experiments/README.md`
